## 🔍 关键词搜索 (keywords)

关键词搜索功能支持通过试卷名称、内容等字段进行模糊匹配，帮助用户快速定位特定试卷。

### 搜索参数

| 参数名 | 类型 | 必需 | 说明 |
|--------|------|------|------|
| `keywords` | string | 否 | 搜索关键词，支持中文和英文 |

### 使用示例

```bash
# 搜索包含"思维"的试卷
down-paper -k "思维" -c "your-cookie"

# 搜索特定讲次
down-paper -k "第4讲" -c "your-cookie"

# 搜索特定题型
down-paper -k "应用题" -c "your-cookie"

# 搜索特定知识点
down-paper -k "分数" -c "your-cookie"
```

### 搜索效果

- **精确匹配**：关键词会匹配试卷名称、描述等字段
- **模糊搜索**：支持部分匹配，提高搜索灵活性
- **多语言支持**：支持中文、英文关键词搜索
- **特殊字符**：自动处理URL编码，支持特殊字符

### 搜索建议

1. **使用具体关键词**：如"第4讲"、"应用题"等
2. **避免过于宽泛**：如"数学"、"测试"等可能返回过多结果
3. **组合使用**：结合年级、学期等参数进一步缩小范围
4. **测试验证**：可以先使用关键词测试，确认结果后再批量下载

## 🖥️ 无头模式 (headless)

无头模式控制浏览器是否显示窗口，适合不同的使用场景。

### 模式参数

| 参数名 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| `headless` | boolean | false | 是否使用无头模式运行浏览器 |

### 使用示例

```bash
# 默认行为：显示浏览器窗口（有头模式）
down-paper -u "khlx" -g "0557" -q "3" -c "your-cookie"

# 无头模式：不显示浏览器窗口
down-paper -u "khlx" -g "0557" -q "3" -c "your-cookie" -h

# 或者使用完整参数名
down-paper -u "khlx" -g "0557" -q "3" -c "your-cookie" --headless

# 显式指定显示浏览器
down-paper -u "khlx" -g "0557" -q "3" -c "your-cookie" --no-headless
```

### 模式对比

| 模式 | 参数 | 效果 | 适用场景 |
|------|------|------|----------|
| **有头模式** | 无参数或 `--no-headless` | 🖥️ 显示浏览器窗口 | 调试、开发、查看页面状态 |
| **无头模式** | `-h` 或 `--headless` | 🚫 不显示浏览器窗口 | 服务器环境、自动化脚本、静默运行 |

### 使用建议

1. **开发调试**：使用有头模式，可以观察页面加载和操作过程
2. **生产环境**：使用无头模式，提高性能并避免界面干扰
3. **服务器部署**：必须使用无头模式，因为服务器通常没有图形界面
4. **自动化脚本**：推荐使用无头模式，确保脚本稳定运行
